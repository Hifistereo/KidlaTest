Burtu Feja — word audio clips
================================

Drop one short mp3 per word into this folder. The game plays audio/<slug>.mp3
when a child wins a level or taps the picture card (the 🔊 button).

Record a clear, friendly human voice saying the Latvian word. Keep each clip
short (~1 second). If a file is missing, the game just stays silent for that
word — no robotic voice, no error. So you can add clips gradually.

Format: .mp3, mono is fine, normal speaking volume.
Filename: exactly the slug below + ".mp3", all lowercase, no diacritics.

Special files (2)
-----------------
background5min.mp3 — looping background music for the whole session.
                     Plays quietly (volume ~0.12) and repeats. Starts on the
                     first tap (browsers block autoplay until then).
win.mp3            — short celebration sound, played on every correct answer
                     (just before the word is spoken).
