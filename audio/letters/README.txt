BURTU FEJA — LETTER SOUND CLIPS (for the "Skaņas" / sound-blending game)
========================================================================

Record one short clip per letter and drop it in THIS folder (audio/letters/).
Record the SOUND the letter makes (the phoneme), NOT the letter's name.
  e.g.  R  -> "rrr"  (not "er"),   S -> "sss",   A -> "ah"

Files that are missing simply stay silent — the game still works, it just
won't play that sound. No text-to-speech is ever used.

Keep clips short (~0.4–0.8 s), trimmed, same voice/volume as the word clips.
Format: mp3.

Filename scheme (ASCII slug per letter):
  - plain letter      = the letter itself        A -> a.mp3
  - macron (garumzīme)= the letter doubled        Ā -> aa.mp3
  - háček / soft mark = letter + "x"              Č -> cx.mp3
